# 明日方舟肉鸽伤害计算器

## 致力于开发一款简单易用的明日方舟肉鸽伤害计算器，帮助玩家更好地理解和优化其角色的伤害输出。

## 功能特点
- **角色选择**：支持选择不同的明日方舟角色进行伤害计算
- **藏品选择**：支持藏品选择和搜索！
- **敌方属性输入**：支持输入敌方防御力、法抗等进行伤害计算
- **伤害计算**: 计算精确到远超游戏内使用精确度的伤害，并提供详细的倍率分析。
- **实现方式**：对于无概率类型技能和概率期望可算的技能，直接计算期望伤害输出； 对于难以计算期望的，采用蒙特卡洛模拟进行伤害计算
> **注意！**： 目前版本没有实现藏品匹配，藏品对当前干员能否生效请自行注意！

double AtkSpeed, Atk[6][MAX], Sum;
//主要的全局变量
//AtkSpeed：攻击速度
//Atk[][]：存储倍率乘区的二维数组，
//         第一维是干员维度，用于干员技能中不同的攻击类型，具体的下文干员数据部分解释。
//         第二维是倍率乘区维度，每一位是一个独立乘区，从0开始编号。
//         在第二维的维度中，同位叠加，不同位叠乘。
//Sum：总伤害

struct AtkSpeedRelic{
    double atkSpeed;
    char name[30];
};
//藏品结构体，直接的第一个成员存数值，倍率按小数方式。第二个成员存名字。
//为了实现匹配功能，后续预期会增加bool[] 类型独热编码，用于匹配生效干员类型

struct Operator{
//干员结构体
    char name[30];
    double atk;//面板攻击力，目前默认全满练度硬编码，后续预期会接入练度算法
    double skillDuration[6];//持续时间
    //统一解释[6]位数组：
    //对于干员在技能中各种奇妙的攻击方式和不同攻击倍率，拆开来一个个计算。长度是6是因为目前拆分最复杂的拆了6个，直接硬编码了，后续会改。
    //非零值表示固定持续时间的攻击类型，零值表示固定次数攻击，具体的次数在atkCount中按照单次连击实现。
    double skillMul[6][4];//面版乘算，直接乘算叠加，最终乘算叠乘，脆弱乘算叠乘
    //[4]中存储干员技能中的4种乘区倍率。
    //注意前两位叠加乘区，默认值0，后两位叠乘乘区，默认值1。
    double atkSpeed;//面板攻击速度
    //有自带天赋、技能加成的干员，在这里直接把天赋加成算进去，上面的面板攻击力同理。
    double atkInterval;//攻击间隔
    //同上，技能天赋效果硬写入。后续加入练度选择时会改。
    double atkCount[6];//攻击次数
    //每次攻击动作的连击次数，可以是正常连击，也可以配合持续时间取零值实现固定次数攻击。
    //注意！有概率的干员，概率触发体现在攻击次数里。例如60%概率的伤害，攻击次数取0.6。
    bool damageType[3];//物理，法术，元素伤害类型
    double ignoreDef[3];//防御力，法抗，元素抗性削减、无视
};
